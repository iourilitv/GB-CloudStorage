Don't change any DEFAULT properties.

You could change some properties in the file "config.json" as far as:
"port_custom":0(default);

Users files are in the "storage"(default).

ATTENTION!!!
The parameter "root_absolute" should be used only with "\\" instead of "\"!
for instance:
"D:\\GeekBrains\\20191130_GB-Разработка_сетевого_хранилища_на_Java\\cloudstorage\\server\\target"
